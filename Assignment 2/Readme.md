# Linear Data Structures Toolkit (LDST)

This project implements basic linear data structures in Python from scratch as part of a Data Structures assignment.

## Features

- Dynamic Array with resizing
- Singly Linked List
- Doubly Linked List
- Stack (using linked list)
- Queue (using linked list)
- Balanced Parentheses Checker

## Concepts Covered

- Array resizing and amortized time complexity  
- Linked list operations  
- Stack (LIFO) and Queue (FIFO)  
- Real-world application using stack  

## How to Run

Run the file using:

```python ldst_toolkit.py```

## Output

The program demonstrates all operations with test cases printed on the console.

## Files Included

- ldst_toolkit.py → main implementation  
- report.pdf → explanation and analysis  
- output.txt → program output
